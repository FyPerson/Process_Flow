## 怎么算"横切问题"

满足任一即可放这里：
- 工具链问题（部署 / 构建 / dev 启动）
- 跨多个阶段反复出现（不只一次）
- 解决方案不属于"阶段功能"语义（不是新增节点权限 / 批注这种功能）
- 修复完后所有阶段受益（不是只服务某个阶段）

不满足时放对应阶段子目录（如 P2H 期间发现的 React 状态 bug 应该放 `阶段2/P2H/`）。
