## 关键架构决策（不重复 01/02/03/04 已写过的细节）

1. **conflict_logs 写入 service 内同事务**（取舍审判断点 1 + 隐藏判断 #6）：与 saveCanvas 主表 / canvas_versions 同事务，插入失败让事务回滚，不静默吞
2. **details 字段含 deltaA/deltaB 完整**（判断点 2）：用 tryMerge includeDebugDelta=true 返回的 debugDelta 序列化；64KB 上限 + truncated 标志
3. **客户端不能 import server，必须类型镜像 + 契约测试守门**（判断点 7）：契约测试用 `Assert<T extends true>` + `IsEqual<X,Y>` 真 compile-fail pattern，删字段会让 tsc 真报错（验证已通过）
4. **B 方案保核心不变量 serverVersionRef ≡ projectRef 服务端基线**（复审戳穿 A 方案后的修法）：merged=true + changeSeq 不等时，server-side state 全不动；下次 save 用旧 baseVersion 让服务端再合并把 A 改动并入；保留本地后续编辑
